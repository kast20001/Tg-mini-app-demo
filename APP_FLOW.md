## Описание работы приложения

### 1. Главная страница (index.html)
- Стартовая страница с логотипом и описанием приложения
- Ссылка на раздел "Тесты" → Tests_prob.html

### 2. Выбор темы (Tests_prob.html)
- Список доступных тем (3 активных)
- Каждая тема ведёт на страницу вопроса с параметром `?question=N`
  - Тема 1 → Task_content.html?question=1
  - Тема 2 → Task_content.html?question=2
  - Тема 3 → Task_content.html?question=3

### 3. Решение задачи (Task_content.html)
- Загружает данные из question_data_mock.js
- Показывает текущий вопрос (номер, текст, варианты ответов)
- **Кнопка "Назад"** → переход на Tests_prob.html (если вопрос 1) или на предыдущий вопрос
- **Кнопка "Вперед"** → переход на следующий вопрос или на результаты (если последний вопрос)
- **Кнопка подсказки** → открывает модальное окно с решением

### 4. Решение с объяснением (task_solution.html)
- Загружает данные из question_data_mock.js по параметру ?question=N
- Показывает текст задачи и полное решение
- **Кнопка "К вопросам"** → возврат на Task_content.html?question=N
- **Кнопка "Заново"** → возврат на Tests_prob.html

### 5. Результаты (out_results.html)
- Показывает итоги тестирования
- Количество правильных ответов (X/10)
- Иконка в зависимости от результата
- Время прохождения
- **Кнопка "К боту"** → на главную (index.html)
- **Кнопка "Тесты"** → на выбор тем (Tests_prob.html)
- **Кнопка "Заново"** → начать сначала (Task_content.html?question=1)

## Структура данных

### question_data_mock.js
```javascript
const questions = [
    {
        number: 1,
        question: "Текст вопроса",
        solution: "Подробное решение"
    },
    // ... больше вопросов
];
```

## Навигация параметрами URL

- `Task_content.html?question=1` - вопрос № 1
- `Task_content.html?question=2` - вопрос № 2
- `Task_content.html?question=3` - вопрос № 3
- `out_results.html` - страница результатов

## Файлы JavaScript

1. **question_data_mock.js** - База данных вопросов и решений
2. **task_content.js** - Логика страницы с вопросом (навигация, расширение ответов)
3. **task_solution_script.js** - Логика страницы с решением
4. **out_results_script.js** - Логика страницы с результатами
